# PowerBI-Dashboard
# Sales Analysis Power BI Dashboard

## Overview
This project is a comprehensive **Power BI Sales Analysis Dashboard** designed to analyze business performance using interactive reports, KPIs, and visual insights.  

The dashboard helps users understand overall sales trends, customer behavior, and product performance through multiple interconnected reports.

The project contains **3 different reports/pages** with navigation between reports for a seamless analysis experience.

---

# Features
- Interactive Power BI Dashboard
- Sales KPI Tracking
- Customer Performance Analysis
- Product Performance Analysis
- Cross-report Navigation
- Dynamic Visualizations and Filters
- Business Insights for Decision Making

---

# Reports Included

## 1. Overview Report
The Overview Report provides a high-level summary of the overall business performance.

### Key Insights
- Total Sales
- Total Profit
- Revenue Trends
- Sales by Region
- Monthly/Yearly Performance
- Category-wise Analysis

### Visuals Used
- KPI Cards
- Line Charts
- Bar Charts
- Pie Charts
- Slicers and Filters
- Maps

---

## 2. Customer Report
This report focuses on customer-related analysis and purchasing behavior.

### Key Insights
- Top Customers
- Customer-wise Sales
- Customer Segmentation
- Purchase Trends
- Repeat Customer Analysis

### Visuals Used
- Customer Ranking Charts
- Donut Charts
- Tables and Matrix Visuals
- Trend Analysis Charts
- Interactive Filters

---

## 3. Product Report
This report provides detailed analysis of product performance and sales contribution.

### Key Insights
- Top Selling Products
- Product-wise Revenue
- Profitability Analysis
- Category and Sub-category Performance
- Product Demand Trends

### Visuals Used
- Bar Charts
- Tree Maps
- KPI Cards
- Product Comparison Visuals
- Interactive Slicers
